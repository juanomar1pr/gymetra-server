Running a Meet with Gymetra

This guide explains the typical workflow used to run a gymnastics competition using Gymetra.

1. Create the Meet

Start by entering the basic information for the competition.

• Enter meet name  
• Enter location  
• Set date and time  

Federation Setup

Before continuing, select the federation type for your meet.

• Select Federation Type
• After selecting a federation, additional options will appear
• Add Judges (based on federation rules)
• Add Coaches
• Enter required federation-specific information
Note: The meet name is also used to generate the public results link. This name will appear in the advertising/public results URL, so choose a clear and appropriate name. Tip: After entering the meet name and date you can use the lock icon 🔒 to lock these fields and prevent accidental changes.
Save or Open a Project

• Save the meet project  
• Open an existing project  
• Continue editing meet information later

2. Federation & Assign Judges

If the meet belongs to a federation, select the federation and assign judges to each apparatus.

• Select federation (if applicable)  
• Assign judges to apparatus  
• Verify judge assignments  
Tip: Filters can help locate judges quickly when assigning them.

3. Judges Connect and Verify Device

After judges are assigned, each judge should open the Gymetra application on their iPad and connect to the host device.

• Judge opens Gymetra on the iPad  
• Device connects to the host session  
• The assigned apparatus appears on screen  
Tip: Judges should confirm the correct event appears before the meet begins.

4. Add or Import Athletes

Athletes can be entered into the meet in different ways.

• Import athletes from a file  
• Enter athletes manually  
• Review athlete information  

5. Assign Gymnasts to Events or Groups

After athletes are added they can be assigned to events or organized into rotation groups.

• Assign gymnast to apparatus  
• Create rotation groups  
• Adjust athlete order  
Tip: Filters can help locate athletes faster when assigning them to events.

6. Print Meet Documents

Before starting the meet you can generate documents for judges and staff.

• Print labels  
• Print attendance list  
• Print score sheets  

7. Create and Confirm the Session

When all meet information has been entered, create the competition session.

• Create session  
• Confirm session details  
• Activate session  

8. Start the Meet

Once the session is activated the competition can begin.

• Start meet timer  
• Start warm-up timer  
• Judges begin scoring routines  

9. Results

As judges enter scores the results update automatically.

• Scores synchronize in real time  
• Rankings update automatically  
• Results can be broadcast live  
Tip: Tap the question mark icon ❓ to open contextual help explaining the current screen.

Gymetra keeps the entire meet organized and synchronized from setup to final results.
