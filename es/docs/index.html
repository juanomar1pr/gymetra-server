Cómo gestionar una competencia con Gymetra

Esta guía explica el flujo de trabajo típico utilizado para gestionar una competencia de gimnasia con Gymetra.

1. Crear la Competencia

Comienza ingresando la información básica de la competencia.

• Ingresar nombre de la competencia
• Ingresar ubicación
• Configurar fecha y hora
Nota: El nombre de la competencia también se utiliza para generar el enlace público de resultados. Este nombre aparecerá en la URL pública/publicitaria de resultados, así que elige un nombre claro y apropiado.

2. Configuración de Federación

Antes de continuar, selecciona el tipo de federación para tu competencia.

• Seleccionar tipo de federación
• Después de seleccionar una federación aparecerán opciones adicionales
• Agregar jueces (según reglas de la federación)
• Agregar entrenadores
• Ingresar información requerida específica de la federación
Consejo: Después de ingresar el nombre y la fecha de la competencia puedes usar el ícono de candado 🔒 para bloquear estos campos y evitar cambios accidentales. Toca el ícono de signo de pregunta ❓ para abrir ayuda contextual explicando la pantalla actual.
Guardar o Abrir un Proyecto

• Guardar proyecto de la competencia
• Abrir un proyecto existente
• Continuar editando la información después

3. Conectar Dispositivos de Jueces

Antes de asignar jueces, asegúrate de que cada iPad esté conectado a la misma red Wi-Fi que la Mac host. Cuando se abre la app de Gymetra, cada iPad muestra automáticamente un código único de 3 caracteres. Ese mismo código aparecerá en la computadora host, facilitando identificar el dispositivo correcto antes de asignar un juez.

• Conectar todos los iPads de jueces al mismo Wi-Fi que la Mac host
• Abrir Gymetra en cada iPad
• Anotar el código de 3 caracteres mostrado en cada pantalla
• Verificar que el mismo código aparezca en la lista del host
• Relacionar cada iPad con el juez correcto usando el código
• Asignar juez y aparato desde la Mac host
Consejo: Puedes hacer clic derecho en cualquier dispositivo conectado en el host para renombrarlo.

4. Federación y Asignación de Jueces

Si la competencia pertenece a una federación, selecciona la federación y asigna jueces a cada aparato.

• Seleccionar federación (si aplica)
• Asignar jueces a aparatos
• Verificar asignaciones
Consejo: Los filtros ayudan a localizar jueces rápidamente.

5. Agregar o Importar Atletas

Las atletas pueden ingresarse de diferentes maneras.

• Importar atletas desde archivo
• Ingresar atletas manualmente
• Revisar información

6. Asignar Gimnastas a Eventos o Grupos

Después de agregar atletas pueden asignarse a eventos o organizarse en grupos de rotación.

• Asignar gimnasta a aparato
• Crear grupos de rotación
• Ajustar orden de atletas
Consejo: Los filtros ayudan a encontrar atletas más rápido.

7. Imprimir Documentos

Antes de comenzar puedes generar documentos para jueces y staff.

• Imprimir etiquetas
• Imprimir lista de asistencia
• Imprimir hojas de puntuación

8. Crear y Confirmar la Sesión

Cuando toda la información esté lista, crea la sesión.

• Crear sesión
• Confirmar detalles
• Activar sesión

9. Preparar el Espacio de Trabajo

Antes de iniciar, Gymetra permite preparar el espacio para máxima visibilidad y eficiencia.

• Usar botones de expandir en Setup o Meet Control
• Aumentar espacio para jueces, atletas o paneles
• Enfocarte en la sección activa más importante
Consejo: Los paneles expandidos son útiles durante competencias en vivo.

10. Abrir la Pantalla del Marcador

Antes de comenzar, puedes preparar la pantalla en vivo para monitores externos, TVs o arena.

• Abrir pantalla desde Meet Control
• O ir a Ventana → Marcador
• Mover el marcador a una segunda pantalla
Consejo: Ideal para audiencia, entrenadores y rankings en vivo.

11. Configurar Ajustes del Meet

Gymetra permite personalización visual y operativa antes de comenzar.

• Cambiar color de fondo de Gymetra Studio
• Actualizar logos o branding
• Elegir logs completos o por sesión
Puedes verificar la conexión actual revisando la dirección IP del host mostrada en Gymetra

12. Iniciar la Competencia

Una vez activada la sesión la competencia puede comenzar.

• Iniciar cronómetro de competencia
• Iniciar cronómetro de calentamiento
• Los jueces comienzan a puntuar rutinas

13. Resultados

A medida que los jueces ingresan puntuaciones, los resultados se actualizan automáticamente.

• Las puntuaciones se sincronizan en tiempo real
• Rankings actualizados automáticamente
• Resultados pueden transmitirse en vivo

Gymetra mantiene toda la competencia organizada y sincronizada desde la configuración inicial hasta los resultados finales.
